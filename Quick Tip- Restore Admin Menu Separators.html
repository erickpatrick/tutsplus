Dica Rápida: Restaura Separadores do Menu Administrativo

O WordPress ganhou um painel administrativo mais moderno, porém, nem todo mundo está tão feliz com algumas das mudanças encontradas. Pessoalmente, amei o estilo minimalista adotado. 

Só há uma coisa que eu sinto falta: A navegação lateraç não tem mais os separadores das diferentes áreas do WordPress.

Nessa dica rápida, mostrei como inspecionar o CSS do painel e fazer com que os separadores voltem a aparecer, com algumas poucas linhas de código.

Encontrando os Estilos

Os separadores que tentaremos restauras, na verdade, ainda estão presentes - só estão transparentes, porém. Clique com o botão direito na área em branco entre Comentárioss and Aparência e inspecione esse elemento.

Você deverá ver o HTML renderizado. Os menus são elementos ul com elementos li para cada item do menu. 

Abra o elemento ul id="adminmenu" role="navigation". Você verá todos os itens do menu envoltos em elementos li, assim como verá um elemento li class="wp-menu-separator" entre eles. Quando clicar neele, você será capaz de ver os estilos e perceberá o CSS para esse item.

#adminmenu li.wp-menu-separator {
  background:   transparent;
  border-color: transparent;
}

É o estilo do nosso separador. Para ser mais específico, é o estilo que precisamos estilizar.

Adicionando Seus Próprios Estilos CSS ao Painel Administrativo

Agora, criaremos uma função que adicionará nossos próprios estilos CSS ao painel administrativo do WordPress. 

Adicione esse código ao arquivo functions.php do seu tema.

function tutsplus_separators() {

}
add_action( 'admin_head', 'tutsplus_separators' );

A primeira parte define a função em si e o gancho add_action adicionará a função ao cabeçalho do painel administrativo do WordPress. No meio, adicionaremos uma função echo para adicionar nosso próprio códogio CSS.

function tutsplus_separators() {
echo '<style type="text/css">#adminmenu li.wp-menu-separator {margin: 0; background: #444;}</style>';
}
add_action( 'admin_head', 'tutsplus_separators' );

Se recarregar suas páginas, os separadores estarão numa cor cinza, especificamente a #444. Também mudei as margens do separador porque achei ele precisava de um certo alinhamento.

Dando Suporte aos Esquemas de Cores Padrão do WordPress

Se você for um perfeccionista, notará que essa cor só combina com os temas preto e azul padrões do painel administrativo do WordPress. Vejamos como podemos fazer para dar suporte a todos os esquemas de cores.

A tag body do painel do WordPress tem várias classes úteis para identificar os diferentes usos. Se você prestar mais atenção, também verá a classe do esquema de cores. A classe padrão é chamada de admin-color-fresh. 

Tente mudar seu estilo em “Usuários” > “Seu Perfil” e veja essa classe mudar. Com essa classe disponível, podemos atribuit a cada esquema, seu próprio separador personalizado. Apenas adicione o nome da classe na frente do seu código css, assim .admin-color-fresh #adminmenu li.wp-menu-separator {background: #444;}

Aqui está o código de como ficará os outros separadores, combinando com seus respectivos esquemas:

function tutsplus_separators() {
 echo '<style type="text/css">
 		#adminmenu li.wp-menu-separator {margin: 0;}
 		.admin-color-fresh #adminmenu li.wp-menu-separator {background: #444;}
 		.admin-color-midnight #adminmenu li.wp-menu-separator {background: #4a5258;}
 		.admin-color-light #adminmenu li.wp-menu-separator {background: #c2c2c2;}
 		.admin-color-blue #adminmenu li.wp-menu-separator {background: #3c85a0;}
 		.admin-color-coffee #adminmenu li.wp-menu-separator {background: #83766d;}
 		.admin-color-ectoplasm #adminmenu li.wp-menu-separator {background: #715d8d;}
 		.admin-color-ocean #adminmenu li.wp-menu-separator {background: #8ca8af;}
 		.admin-color-sunrise #adminmenu li.wp-menu-separator {background: #a43d39;}
       </style>';
}
add_action( 'admin_head', 'tutsplus_separators' );

Bônus

If you want separator lines between every menu item try tweaking the menu-top class of the li-items a bit with border-bottom and border-top.
